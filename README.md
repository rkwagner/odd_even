## Synopsis
 
The following program takes as input an integer and outputs
whether it is odd or even.

## Example

gcc -o odd odd.c

./odd.exe
Enter a number to check if it is odd or even: 155
155 is odd.

./odd.exe
Enter a number to check if it is odd or even: 156
156 is even.
